# Overview
This project investigates solving the coordination problem in sequential social dilemmas (SSDs) by generating and applying constraint schedules that are generated by an LLM. Specifically, the objective is for the LLM to generate constraint schedules that are robust against schedule violations, ɛ. 

This project uses a modified version of the commons_harvest__open substrate to investigate the capability of large language models (LLMs) to do this in the form of constraint generation. Using text description of the game and relevant information, we can leverage the demonstrated ability of LLMs to think strategically to model high level meta-strategies, which can then be plugged in directly to the simulation framework and tested.

# Results
The naive 'sacred apples' constraint schedule is outclasses by the core protection constraint schedules generated by Claude and ChatGPT. This demonstrates the point that LLMs are able to understand and implement directed and specificstrategies that can manipulate multi-agent behaviour to maximise social welfare in all ɛ tested.
